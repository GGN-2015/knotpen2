# 界面使用手册

[English version](./interface-user-manual.md)

本文说明如何通过图形界面使用 Knotpen2。

## 窗口布局

窗口分成两个功能区域：

- 画布区域，用于绘制扭结和链环图。
- 右侧命令面板，圆角衬底上放置所有操作按钮。

只要点击落在右侧命令面板的圆角衬底内，即使没有点中具体按钮，也会被视为 UI 点击，不会传递到画布。这可以避免误创建节点。

软件启动时会根据当前屏幕大小自适应窗口尺寸。之后可以使用 **增大窗口** 和 **减小窗口** 调整窗口大小，程序会保证窗口不超过屏幕范围。

## 语言与帮助

- **切换语言** 会在英文和中文界面之间切换之后显示的按钮和帮助文字。
- **帮助** 会打开应用内帮助页。
- **关闭帮助** 会关闭帮助页并回到编辑界面。

消息区中已经显示过的历史消息不会重新翻译。

## 视图移动

使用以下按钮平移已有图形：

- **上移视图**
- **下移视图**
- **左移视图**
- **右移视图**

如果当前没有节点，移动视图不会产生可见效果。

## 普通编辑模式

Knotpen2 启动后默认进入普通模式。

- 左键点击画布空白处创建节点。
- 拖拽节点移动节点。
- 右键点击节点删除节点。
- 右键点击边删除边。
- 双击边的中点附近，将一条边拆分为两条相连边。
- 点击两条边的交叉点，切换哪条边绘制在上方。

度不等于 2 的节点会显示为实心灰色。计算 PD_CODE 前必须修复这些节点。

## 连续创建模式

点击已有节点可以选中它。选中节点会变红，程序进入连续创建模式。

在此模式下：

- 左键点击空白处会创建新节点，并自动连接到当前选中节点。
- 新节点会成为新的红色选中节点。
- 点击另一个已有节点可以闭合曲线，并退出连续创建模式。
- 左键点击红色选中节点，或者在命令面板外右键点击任意位置，可以退出连续创建模式。
- 点击 **删除选中点** 可以删除当前选中节点，并回到普通模式。

## 起始点与方向点

计算 PD_CODE 前，每个连通分支都必须恰好有一个起始点和一个方向点。

设置方法：

1. 点击节点并选中它。
2. 点击 **设为起始点**，该节点会以蓝色标记。
3. 点击另一个相邻节点。
4. 点击 **设为方向点**，该节点会以绿色标记。

起始点和方向点必须位于同一连通分支，且必须相邻。

## 清空与恢复

- **清空全部** 会先创建自动备份，然后清空所有图形数据。
- **恢复存档** 会读取最近一次自动保存文件。

自动保存文件位于被执行程序同目录的 `auto_save` 文件夹。

## PD_CODE 计算

点击 **计算 PD_CODE** 前，请确认：

- 每个节点恰好有两条相连边。
- 每个连通分支都有一个起始点和一个方向点。
- 起始点和方向点相邻。
- 没有三线共点。目前程序不会自动检查这一点。

计算成功后，输出会写入被执行程序同目录的 `answer` 文件夹。

生成文件包括：

- `.txt` 文件，包含 PD_CODE 四元组列表。
- `.num.svg` 文件，带弧线编号。
- `.nonum.svg` 文件，不带弧线编号。
- `.arrow.svg` 文件，带方向箭头。
